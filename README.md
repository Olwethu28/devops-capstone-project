# devops-capstone-project

![Build Status](https://github.com/Olwethu28/devops-capstone-project/actions/workflows/ci-build.yaml/badge.svg)

This project is a customer accounts microservice for an e-commerce platform. The service provides REST API endpoints to create, read, update, delete, and list customer accounts. The application will be containerized using Docker and deployed to Kubernetes following DevOps and Agile development practices.
